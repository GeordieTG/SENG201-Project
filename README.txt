IMPORTING THE PROJECT INTO ECLIPSE:

First, extract the zip folder and save it into your eclipse workspace. You should have a folder with its name being cha216_ggi28_TussleOfTribes.

Next, open eclipse and click file -> Import, select General->Existing Projects into Workspace, then Next. Open the imported folder from your workspace and 
click finish. You should now see the project in your project explorer. 

Click on the project, then click the run button. Select Java Application, and press ok. Then, select title screen TitleScreeen - gui, and press ok.
This will now run the application.


To run the jar file:
Open the downloaded folder in a terminal window
Type 'java -jar cha216_ggi28_TussleOfTribes.jar'
